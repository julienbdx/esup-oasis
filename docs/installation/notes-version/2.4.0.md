# Version 2.4.0

Cette version est une refonte technique majeure du frontend. Les modifications sont détaillées dans
le [changelog](../../frontend/CHANGELOGS/CHANGELOG_2.4.md).

Elle apporte également des montées de version de dépendances techniques (frankenphp, api platform...), quelques
corrections de bugs et quelques évolutions.

## Principaux changements

### Changements bloquants

Aucune modification bloquante par rapport à la version 2.3.

### Changements non-bloquants

* Filtrage des bénéficiaires par date de validité du profil.
* Upload de plusieurs fichiers simultanément dans les demandes.
* Onglet "Toutes les campagnes" pour les administrateurs
* Persistence des filtres (Demandeurs, Bénéficiaires, Aménagements, Intervenants) lors de la navigation entre
  les pages.
* Meilleure gestion des exports CSV de grande taille
* Resserrement de la sécurité des accès API à certaines ressources (photos...)
* Ajout d'un mode antivirus strict pour la gestion des fichiers téléversés.

## Migration depuis une version 2.3

Plusieurs variables d'environnement ont changé :

* REACT_APP_VERSION : supprimée au profit d'une injection automatique lors du build.
* REACT_APP_GERER_DEMANDES : nouvelle variable permettant de désactiver la gestion des demandes pour les établissements
  qui n'en ont pas l'usage.
* REACT_APP_MAX_FILE_SIZE : taille max des fichiers en upload (en Mo) - 10Mo par défaut
* CLAMAV_STRICT_MODE : nouvelle variable permettant de refuser strictement les téléversements en cas d'indisponibilité
  de l'antivirus - true par défaut.
